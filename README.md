## Features

### Expense Visualization:

- Represent expenses using interactive line charts and bar graphs.
- Visualize spending trends over time for better financial insights.

### Dark Mode:

- Enable a dark mode option for improved visibility and reduced eye strain, especially in low-light environments.

### Expense History:

- Maintain a comprehensive history of all expenses logged within the app.
- Easily access and review past expenditures for budgeting and planning purposes.


### Change Password:

- Allow users to securely change their account passwords within the app.
- Enhance account security and user control over their personal information.

### Forgot Password with Email:

- Implement a password recovery feature that enables users to reset their passwords via email.
- Provide a seamless and secure method for users to regain access to their accounts in case of forgotten passwords.

## Technologies Used:

### Frontend:

- React: A JavaScript library for building user interfaces.
- Next.js 13: A React framework for server-rendered applications.
- TypeScript: A statically typed superset of JavaScript that compiles to plain JavaScript.
- shadcn/ui: A UI library used for building stylish and responsive user interfaces.

### Backend:

- Node.js: A JavaScript runtime environment for executing JavaScript code server-side.
- Next 13 API routes
- TypeScript: Used for backend development to add static typing and other advanced features to JavaScript.

### Database:

- PostgreSQL
- Supabase

### other tools and libraries :

- Nodemailer : for sending reset password link.
- BcryptJs : for securing user password.
- jsonwebtoken : generating token for athorization.
